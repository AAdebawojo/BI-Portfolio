Skills

Data Visualisation

I leverage data visualisation tools like Power BI, Tableau, and Looker to transform raw data into insightful graphical representations. By creating dynamic dashboards and reports, I ensure that stakeholders can easily understand complex data, allowing for quick, data-driven decisions. My focus is on incorporating interactive elements that enable real-time exploration of insights, making data not just accessible but compelling.

Programming Language

I utilise programming languages such as Python and SQL to enhance my data analytics capabilities. SQL is integral to my workflow, allowing me to query and manipulate large datasets efficiently within relational databases. Complementing this, I harness Python's extensive libraries like Pandas and NumPy for complex data manipulations, statistical analysis, and automation of data processing tasks. This combination enables me to conduct advanced analyses and deliver deeper insights that drive business success.

Data Cleaning & Wrangling

I prioritise data cleaning and wrangling as essential steps in my analytics process, using tools such as Azure Data Factory, FME, Excel, and Power Query. I meticulously prepare and structure data by identifying inconsistencies and handling missing values, ensuring that it meets high-quality standards. This preparation is crucial as it sets the foundation for accurate analysis and visualisation, enabling me to transform messy data into a usable format for impactful insights.

Data Governance

I recognise the importance of data governance in maintaining data integrity, security, and compliance. I implement robust governance protocols, to establish standards and policies for data management. By doing so, I ensure that the analytics insights I provide are based on trustworthy data, fostering confidence among stakeholders and supporting effective decision-making throughout the organisation.
